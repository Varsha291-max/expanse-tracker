# expanse-tracker
Expense Tracker Project
📌 Overview
This project is a simple Python program that helps users track their daily expenses. It allows adding expenses with categories, viewing all expenses, and calculating total spending.

🎯 Features
Add expenses with category
Display all expenses
Show total spending
🛠️ Concepts Used
Lists
Loops
Calculations
⚙️ Requirements
Python 3.x installed
🚀 How to Run
Clone the repository:
